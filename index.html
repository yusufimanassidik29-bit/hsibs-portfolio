Tentang Saya

Latar Belakang & Visi

Saya adalah seorang Frontend Developer sekaligus santri aktif yang berdomisili di Indonesia. Keseharian saya diisi dengan menuntut ilmu agama di pesantren, yang sekaligus membentuk cara pandang saya dalam membangun kode: terstruktur, beradab, dan bertujuan membawa manfaat.

Dunia desain grafis dan coding adalah wadah kreativitas saya. Saya percaya bahwa teknologi terbaik adalah teknologi yang tidak hanya canggih secara fungsional, tetapi juga bersih dan menenangkan saat dipandang pengguna.

Fokus utama pengembangan saya saat ini adalah menguasai ekosistem pengembangan web modern menggunakan HTML5, Tailwind CSS, Javascript, serta integrasi sisi server berbasis Python framework.
